# 🐣 구현해야 하는 기능 목록

## 문제 1

### 점수 계산 함수 구현
- 페이지 번호의 각 자리 숫자를 모두 더한다.
- 페이지 번호의 각 자리 숫자를 모두 곱한다.
- 가장 큰 수를 return 한다.

### 포비의 점수 계산
- 점수 계산 함수에 포비의 왼쪽 페이지 번호를 넣어 점수를 계산한다.
- 오른쪽 페이지 번호도 동일하게 점수를 계산한다.
- 둘 중 큰 점수를 포비의 점수로 한다.

### 크롱의 점수 계산
- 크롱의 점수도 포비의 점수와 동일하게 진행한다.

### 점수를 비교한다.
- 포비가 높으면 1
- 크롱이 높으면 2
- 같으면 0
- 예외는 -1
